## ✨ What's This?

A sleek, modern single-page website for **TheoneexLabs** — a DevOps & SRE consultancy. Built with pure HTML, CSS, and JavaScript. No frameworks, no bloat.

## 🎯 Sections

| Section | Highlights |
|---------|-----------|
| **Hero** | Animated terminal graphic, auto-sliding tech marquee |
| **About** | Company story + key stats |
| **Services** | 6 service cards with hover effects |
| **Pricing** | 3-tier pricing cards (Starter / Growth / Enterprise) |
| **Contact** | Working contact form via [FormSubmit.co](https://formsubmit.co) |

## 🛠️ Tech Stack

- **Frontend:** Vanilla HTML5, CSS3, JavaScript
- **Font:** [Inter](https://fonts.google.com/specimen/Inter) via Google Fonts
- **Icons:** [Devicon](https://devicon.dev) CDN (AWS, Docker, K8s, Terraform, Ansible, Jenkins, GitLab, GitHub)
- **Form Backend:** FormSubmit.co (with `fetch` API + `mailto:` fallback)

## 🚀 Getting Started

```bash
# Just open it
open index.html

# Or serve it locally
npx serve .
```

No build step. No `npm install`. Just open and go.

## 📬 Contact Form

The form submits via **FormSubmit.co** using the `fetch` API. If submission fails (e.g., offline or CORS), it falls back to a `mailto:` link — so the user's message is never lost.
